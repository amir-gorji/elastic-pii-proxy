- For a sample MCP setup and configuration details, see `.github/mcp.json` which has been made available outside of gitignore for reference.

To build the MCP server, run:

```bash
npm run build:mcp
```

The following npm scripts are available:

- `npm run build:mcp` - Builds the MCP server
- `npm start` - Starts the application
- `npm test` - Runs the test suite
